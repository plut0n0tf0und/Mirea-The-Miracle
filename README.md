# ✦ Mirea — The Miracle ✦  
> *A retro-social space for friends to vote, chat & vibe.*

---

## 🌌 Concept
Mirea is not just an app — it’s a **group ritual**.  
Friends gather, propose events, vote on them, and chat in a neon-lit retro lounge.  
Every face is unique, every card is a choice, every chat line glows.

---

## 🃏 Cards & Polls
- Create **event cards** with details.  
- Friends **vote** to decide what’s next.  
- All events live in a **neon datastore** — glowing, persistent, collective.

---

## 💬 Retro Chatbox
- A **group chat** styled like old-school terminals.  
- **Intuitive & interactive** — feels alive, not static.  
- Each user gets a **retro avatar face** — quirky, distinct, nostalgic.

---

## ✨ Features
- 🗳️ Event voting system  
- 🎨 Neon retro UI  
- 👤 Unique avatars per user  
- 💾 Persistent event storage  
- 💬 Real-time group chat  

---

## 🔮 Why Mirea?
Because planning with friends should feel like a **game**, not a chore.  
Because retro aesthetics remind us that **community is timeless**.  
Because every miracle is just friends deciding together.

---

## 🚀 Quick Start
1. Open the app.  
2. Add your event card.  
3. Vote or chat with your group.  
4. Watch the neon glow as decisions unfold.
